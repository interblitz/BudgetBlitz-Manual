Импорт данных¶

Настройки импорта уведомлений¶

Настройки импорта уведомлений играют важную роль в процессе импорта уведомлений. Если банк меняет структуру уведомлений, то вместе со структурой меняются и настройки импорта. В этом случае Вы можете загрузить обновление настроек или отредактировать настройки самостоятельно, см. главу Расширенная настройка импорта SMS и push-уведомлений.

Для загрузки обновлений выберите пункт меню Действия ‣ Импорт ‣ Настройки импорта SMS и Push.

Приложение покажет доступные обновления. Здесь же можно загрузить настройки для новых банков.

Возможно, что приложение не покажет доступные настройки импорта уведомлений. В этом случае проверьте, что в настройках включен обмен настройками импорта SMS.

SMS и Push уведомления¶

По умолчанию Блиц Бюджет для Android автоматически импортирует SMS и Push уведомления. Тем не менее, в приложении есть возможность в любой момент импортировать вручную SMS и push-уведомления. Для этого откройте справочник SMS сообщения, Push уведомления или специальный диалог импорта:

  1. Откройте диалог импорта.

  2. Выберите счет, для которого нужно импортировать SMS. В счете должны быть указаны идентификатор и настройка импорта SMS.

  3. Отметьте галочками SMS для импорта.

  4. Нажмите Импорт. Кнопка будет доступна, если есть отмеченные SMS.

  5. Проверьте результат в списке операций.

  6. Проблемы, возникшие при импорте, можно увидеть в журнале событий.

CSV файлы¶

Приложение Блиц Бюджет для Android может загрузить из файлов в формате CSV операции и справочники Счета, Статьи, Плательщики и получатели, Проекты, Персоны и Места. Справочники могут быть загружены как из файла с операциями, так и из отдельного файла.

Программа автоматически определяет разделитель колонок, который может быть одним из символов «;», «,», «|», «/», «\».

Первая строка файла должна содержать имена колонок, регистр не имеет значения. Помимо этого, имена колонок могут быть заданы в любой другой строке, тогда они будут иметь силу для последующих строк.

Имена колонок должны быть в списке имен, которые поддерживает приложение. Колонка, имя которой не поддерживается приложением, игнорируется. Для справочников и операций списки различаются, и приведены ниже.

Справочники¶

Возможен импорт справочников Счета, Статьи, Плательщики и получатели, Проекты, Персоны и Места. Поддерживается импорт древовидных данных, для этого полный путь к элементу должна быть указан в колонке Name с разделителем «/», «\» или другим. Сам разделитель можно указать в диалоге импорта. Например, из строки:

Расходы/Прочие

будут созданы группа Расходы и элемент Прочие, принадлежащий группе Расходы.

Поддерживаются имена колонок (если в таблице указано несколько имен, то в файле должно использоваться только одно из перечисленных в таблице):

Операции¶

Возможен импорт фактических и плановых операций, сплитов, а также переводов. В последнем случае из одной строки файла создаются две операции. Названия элементов справочников можно указывать с учетом иерархии, например:

Расходы/Прочие

При импорте операции с такой категорией дополнительно будут найдены или созданы группа Расходы и элемент Прочие, принадлежащий группе Расходы.

По-умолчанию, без изменений CSV файла поддерживается импорт файлов, созданных в приложениях:

  • AbilityCash

  • AnMoney

  • Bluecoins

  • Financisto

  • Дзен-мани

  • Handy Money

Поддерживаются имена колонок (если в таблице указано несколько имен, то в файле должно использоваться только одно из перечисленных в таблице):

OFX файлы¶

Блиц Бюджет для Android поддерживает импорт OFX файлов. Поддерживается спецификация начиная с версии 2.1.1. Для импорта:

  1. Откройте диалог импорта.

  2. Выберите файл для импорта.

  3. Нажмите Далее и отметьте галочками строки для импорта.

  4. Нажмите Импорт. Кнопка будет доступна, если есть отмеченные строки.

  5. Проверьте результат в списке операций.

  6. Проблемы, возникшие при импорте, можно увидеть в журнале событий.

Электронные чеки¶

Блиц Бюджет для Android поддерживает импорт электронных чеков. Возможен импорт чеков:

  1. В формате JSON из приложения Проверка чеков ФНС России.

  2. http-ссылки на чеки в формате HTML в сервисе platformaofd.ru (авторизация не требуется).

  3. Из текста электронного письма Ozon с чеком. Возможно объединение чеков по одному заказу.

  4. Из QR кода на бумажном чеке. В этом случае будет создана операция на сумму и дату, указанные в QR коде. QR код необходимо сканировать и распознать в любом стороннем приложении.

  5. В формате XML из личного кабинета на cabinet.tax.gov.ua для Украины.

Для импорта из сторонних приложений:

  1. Откройте стороннее приложение.

  2. Выберите нужные данные, нажмите Поделиться / Отправить / Передать или другую подобную по смыслу кнопку.

  3. В качестве получателя укажите Блиц Бюджет для Android.

  4. Далее следуйте инструкциям на экране.

Для импорта из буфера обмена:

  1. Откройте диалог импорта.

  2. Скопируйте данные из буфера обмена в диалог.

  3. Далее следуйте инструкциям на экране.

Проблемы, возникшие при импорте, можно увидеть в журнале событий.
